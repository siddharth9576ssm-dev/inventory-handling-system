# Inventory Handling System

A simple inventory management web app built with HTML, CSS, JavaScript, and a Node.js server.

## Features

- User signup and login
- Add, edit, and delete inventory products
- Track product quantity, price, marketplace, and product links
- Dark mode support
- Profile and contact pages
- Backend API using Node.js
- Local JSON database for demo users and inventory storage

## Project Files

- `public/index.html` - Main web page
- `public/style.css` - App styling
- `public/app.js` - Frontend logic
- `server.js` - Node.js backend server
- `localdb.json` - Local private database created when the app runs

## How to Run

Make sure Node.js is installed, then run:

```bash
node server.js
```

Open the app in your browser:

```text
http://localhost:3000
```
